Counts and Tallies of Variables in a dataset

Sometimes it is convenient to to have a total tally() (denominator) of your dataset, or by current groups. This helps in calculating proportions or percentages. You can also perform counts by groups and subgroups. The simplest way to count is with the n() function, which counts the observations and places the result in a vector named (by default) ‚Äún‚Äù.

Exercise 1

Write the R code required to tally the total number of observations with thetally() function, and name the new variable ‚Äúobservations‚Äù.

laryngoscope %>% 
  ---(name = "---")
laryngoscope %>% 
  tally(name = "observations")

Exercise 2

If your data are currently grouped, tally() will count observations by group.

Write the R code required to (1) group_by() ASA class (‚Äúasa‚Äù), then (2) tally the total number of observations in each group with thetally() function, and name the new variable ‚Äúobs‚Äù.

laryngoscope %>% 
  ---(asa) %>% 
  ---(name = ---)
laryngoscope %>% 
  group_by(asa) %>% 
  ---(name = ---)
laryngoscope %>% 
  group_by(asa) %>% 
  tally(name = "obs")
