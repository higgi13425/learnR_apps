Write the R code required to rename the variable fam_hx to fhx. Remember (new_name = old_name) syntax.

  rename(--- = ---)
prostate %>% 
  rename(fhx = fam_hx)

Write the R code required to rename all of the variables that start with ‚Äúpreop‚Äù by replacing ‚Äúpreop‚Äù with ‚ÄúPRE‚Äù, using replace_with(), and the str_replace() function

prostate %>% 
  rename_with(--- , ---)
prostate %>% 
  rename_with(--- , ---)
prostate %>% 
  rename_with(~str_replace(., "preop", "PRE_"),
              starts_with("preop"))
